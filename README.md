🚀 Anime Odyssey Hub

📌 Overview
Anime Odyssey Hub is an intelligent anime recommendation system that helps users discover new anime based on their preferences. It combines machine learning, voice search, and interactive visualizations to deliver a personalized experience.

Built using Streamlit, the app provides a clean UI where users can search anime by text or voice, filter by genres, and explore recommendations with detailed insights.

🎯 Features
🔍 Smart Recommendations using TF-IDF + Nearest Neighbors

🎤 Voice Search using Speech Recognition

🎭 Genre-Based Filtering

📊 Interactive Visualizations with Plotly

🔎 Fuzzy Search for typo-tolerant queries

⚡ Fast and responsive UI with Streamlit

🌐 API integration for anime details

🛠️ Tech Stack
Python

Streamlit – Web UI

Pandas – Data handling

Scikit-learn – Recommendation engine

TF-IDF Vectorizer – Text feature extraction

Nearest Neighbors – Similarity model

SpeechRecognition – Voice input

FuzzyWuzzy – Smart search matching

Plotly – Data visualization

Requests – API calls

🧠 How It Works
Anime dataset is processed using TF-IDF vectorization

Nearest Neighbors model finds similar anime

User inputs:

Text search OR

Voice input

Fuzzy matching improves search accuracy

System returns top recommendations

Visual insights are displayed using Plotly
